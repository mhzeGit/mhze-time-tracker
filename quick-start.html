Saving Your Data

Important: Your data is stored in JSON files on your computer. Remember to save your work!
1

Click "Save Data" in the app

Choose a location and filename (e.g., my-time-tracker.json)

2

Keep the file safe

Store it in a memorable location like:

  • Documents folder
  • Cloud storage (Dropbox, Google Drive)
  • Version control (Git)
3

Load it next time

Click "Load Data" and select your saved file to continue where you left off.

Key Features

  • Add/Edit/Delete: Full control over your time entries
  • Automatic Duration: Calculates time spent automatically
  • Analytics: Daily and weekly averages
  • Charts: Visual representation of your time
  • Responsive: Works on desktop and mobile
  • Offline: No internet required (after initial load)
  • Private: All data stays on your device

Best Practices

  • Track time consistently every day
  • Use consistent Type names (Development, Meeting, Research)
  • Save your data file after each session
  • Backup your data file regularly
  • Review weekly analytics to improve productivity
  • Be honest about time spent

Privacy & Security

Your data is 100% private:
  • No backend server
  • No data sent anywhere
  • All processing happens in your browser
  • You control where data is stored
Remember: Don't commit your actual data file to public repositories! Use .gitignore to exclude it.
